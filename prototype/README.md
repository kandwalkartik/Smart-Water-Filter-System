# Prototype

The primary document describes a breadboard/laboratory prototype and includes figure captions for prototype experimental setup photographs.

## Documented Setup

The prototype includes:

- ESP8266 NodeMCU controller
- DFRobot Gravity TDS-10 sensor
- Relay module
- RO and UF solenoid-valve routing paths
- Push-button mode selection
- LED mode indication
- Firebase and Flutter based data logging/display
