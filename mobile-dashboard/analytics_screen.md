# Analytics Screen

The source document describes an Analytics screen in the Flutter app.

## Displayed Metrics

- Energy consumed in kWh
- Energy saved in kWh
- Savings score versus always-RO baseline
- TDS trend chart for the last 12 measurement updates
- Lifetime usage chart over the last 15 days
- UF runtime
- RO runtime
- Water saved

## Example Values From Source Figures

The primary document includes the following figure-level examples:

| Figure Context | Values |
| --- | --- |
